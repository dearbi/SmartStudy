# PPT 制作提示词 — 计算机设计大赛·软件应用与开发方向

## 项目名称：智学伴侣（SmartStudy）—— AI 驱动的沉浸式智能学习平台

---

## 一、核心信息（请严格按此制作 PPT）

### 1. 项目定位
智学伴侣是一款面向中高考、考研考公学生的**AI 驱动的沉浸式智能学习平台**。通过融合**视频学习、AI 辅助分析、数据可视化、任务管理**四大核心能力，解决学生在自主学习场景中专注力分散、进度不可追踪、知识点掌握不清晰的核心痛点。

### 2. 目标用户
- 中学生（中考/高考备战）
- 大学生（考研备考）
- 公考考生
- 自学者/终身学习者

### 3. 技术栈
| 层级 | 技术选型 |
|------|----------|
| 前端框架 | React 18 + TypeScript 5 |
| 构建工具 | Vite 6 |
| 样式方案 | TailwindCSS 3 + PostCSS |
| 数据可视化 | Recharts（雷达图、热力图） |
| Markdown 编辑器 | @uiw/react-md-editor |
| 视频播放 | react-player（支持本地/B站/URL） |
| 状态管理 | Zustand + React Context |
| 后端 | Express + TypeScript（RESTful API） |
| 存储 | JSON 文件持久化 |
| 路由 | React Router v7 |

## 三、设计规范（请严格遵循）

### 色彩系统
| 用途 | 色值 | 说明 |
|------|------|------|
| 主背景 | `#0a0a0a` | 纯黑背景 |
| 卡片背景 | `#111111` | 深色卡片 |
| 主品牌色 | `#D4AF37` → `#F4C430` | 金色渐变 |
| AI 色 | `#A855F7` → `#3B82F6` | 紫色渐变 |
| 高亮色 | `#22C55E` | 绿色（进度/完成） |
| 次级高亮 | `#3B82F6` | 蓝色（数据/分析） |
| 文字主色 | `#FFFFFF` / `#9CA3AF` | 白/灰 |

### 字体
- 中文：思源黑体 / Noto Sans SC
- 英文：Inter 或系统无衬线字体
- 标题加粗，正文常规

### 排版
- 标题字号建议：封面主标题 48pt+，页标题 32pt，小标题 20pt，正文 14-16pt
- 行距 1.5 倍
- 每页文字不超过 7 行，配合图表展示

### 风格要求
- 整体风格：**科技感 + 高端感**，深色背景为主
- 使用毛玻璃（Glassmorphism）效果
- 适度使用渐变和发光效果
- 图标使用 Lucide 风格（简洁线性）
- 避免大段文字，多用图表、截图、架构图
- 每页右下角可加小型页码
