# 智学伴侣（SmartStudy）

> **AI 驱动的沉浸式智能学习平台**  
> 重塑你的学习体验

---

## 一、项目背景

在线学习已成为当代学生备考的主流方式，但普遍存在三大核心痛点：

- **专注力分散** — 线上学习缺乏有效监督，容易被外界干扰
- **进度不可追踪** — 缺乏可视化的数据反馈，难以科学评估学习效果
- **目标不清晰** — 学习任务管理混乱，优先级不明，复习计划缺乏科学依据

智学伴侣（SmartStudy）应运而生，旨在通过 **AI 技术 + 数据可视化 + 科学记忆算法** 三位一体的方式，为备考学生打造一个沉浸式、可追踪、高效率的智能学习环境。

## 三、核心功能

### 1. 沉浸式学习舱

- **双屏同步学习** — 左屏视频播放 + 右屏 Markdown 笔记，实时同步操作
- **智能视频播放** — 支持 URL 加载、本地上传、B站嵌入三种方式
- **专注模式** — 一键切换，隐藏干扰元素，双层进度条追踪专注/暂停时长
- **AI 学习助手** — 自动分析视频生成结构化摘要，内嵌 AI 对话窗口支持实时问答

### 2. 学习数据仪表盘

- **能力维度雷达图** — 四维评估：知识点覆盖度、专注稳定性、测验准确率、复习频率
- **学情热力图** — 仿 GitHub 贡献墙风格，支持年/月/周三级切换
- **核心统计卡片** — 累计专注时长、完成任务数、连续打卡天数、综合掌握度
- **学习资讯 Feed** — 按备考目标分类推送学习资讯

### 3. 动态任务看板

- **多分类管理** — 按中考/高考/考研/考公分类，每类独立看板
- **智能排序** — 按优先级（高/中/低）+ 截止日期自动排序
- **AI 推荐** — 顶部 Banner 展示 AI 推荐任务，支持一键定位
- **进度追踪** — 进度条 + 完成计数实时反馈

### 4. 艾宾浩斯背诵计划

- **遗忘曲线算法** — 基于 1-2-4-7-15 天复习间隔模型
- **频率分类** — 高频/中频/低频词分类管理，自动计算复习日期
- **今日任务仪表盘** — 新学 + 复习任务汇总，每日打卡追踪
- **统计概览** — 学习天数、总卡片数、复习完成率等数据

### 5. 全局笔记管理

- 所有学习舱笔记自动汇总
- 支持搜索、标签过滤、Markdown 渲染
- 一键跳转到笔记详情

## 六、设计特色

- **暗色沉浸主题** — 深黑主背景（#0a0a0a），金色品牌色（#D4AF37），科技感十足
- **毛玻璃卡片** — 卡片式布局，毛玻璃效果，层次分明
- **神经网络动效** — 首页粒子动效，营造 AI 科技氛围
- **流畅动画** — auto-animate 驱动的列表过渡，split-pane 拖拽交互
- **响应式适配** — 适配不同屏幕尺寸，核心功能在 1366×768 以上分辨率完整展示

## 八、应用场景

### 场景一：考研备考
观看名师视频时边看边记，AI 自动生成视频摘要，仪表盘追踪各科复习进度。

### 场景二：高考冲刺
任务看板管理各科复习计划，AI 推荐当前最需完成的任务，艾宾浩斯计划巩固记忆。

### 场景三：英语学习
艾宾浩斯计划管理单词背诵，热力图反馈每日学习活跃度，保持学习连续性。
