## 六、开源组件使用说明

### 6.1 遵循的许可证类型

本项目使用的开源组件主要遵循以下许可证：

**MIT 许可证**（绝大多数依赖）
- 允许自由使用、复制、修改、合并、出版发行、散布、再授权及销售
- 要求保留版权声明和许可声明
- 本项目已在源代码中保留所有上游组件的版权声明

**Apache-2.0 许可证**（TypeScript）
- 允许商业使用、修改、分发、专利授权
- 要求保留版权声明和许可声明
- 要求注明修改内容

**ISC 许可证**（lucide-react）
- 与 MIT 基本一致，允许自由使用和修改
- 要求保留版权声明

### 6.2 合规性说明

- 所有组件均通过 npm 包管理器获取，来源为 npm 官方注册表
- 项目未修改任何开源组件的源代码，所有组件均以"原样"（as-is）使用
- 所有组件的许可证信息可在各自的 `package.json` 及 `LICENSE` 文件中查阅
- 项目本身未分发或二次分发任何开源组件
